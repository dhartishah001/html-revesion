HTML Interview Questions

Top HTML Interview Questions and Answers

1. What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure web pages.

2. What is a tag in HTML?

A tag is a special keyword enclosed in angle brackets used to define an element. Example: <p> for paragraph.

3. What is an element in HTML?

An element in HTML consists of a start tag, content, and an end tag. Example: <h1>Hello</h1>.

4. What is the purpose of the <head> tag?

The <head> tag contains metadata, links to stylesheets, scripts, and title for the HTML document.

5. What is the difference between block and inline elements?

Block elements take full width and start on a new line (e.g., <div>, <p>), while inline elements only take up as much width as needed and flow with text (e.g., <span>, <a>).

6. What are HTML attributes?

Attributes provide additional information about HTML elements. They are always specified in the start tag. Example: <a href="url">Link</a>.

7. What is the <title> tag used for?

The <title> tag defines the title of the document, shown in the browserâ€™s title bar or tab.

8. What is the purpose of the <meta> tag?

The <meta> tag provides metadata like character encoding, author, or viewport settings.

9. How do you insert a line break in HTML?

Use the <br> tag to insert a line break in the content.

10. What are void elements?

Void elements are self-closing tags that do not require a closing tag, like <br>, <img>, <input>.

11. How do you insert an image in HTML?

Use the <img> tag with the src and alt attributes. Example: <img src="image.jpg" alt="description">.

12. What is the difference between <ol> and <ul>?

<ol> creates an ordered list (numbered), while <ul> creates an unordered list (bulleted).

13. What is the use of the <a> tag?

The <a> tag is used to define hyperlinks. It uses the href attribute to specify the link's destination.

14. What is the use of the <form> tag?

The <form> tag is used to create HTML forms for user input.

15. How do you create a text input field in a form?

Use the <input type="text"> element within a <form>.

16. What is the purpose of the <label> tag?

The <label> tag defines a label for an <input> element, improving form accessibility and usability.

17. What are semantic tags?

Semantic tags clearly define the meaning of the content. Examples include <header>, <footer>, <section>, and <article>.

18. How do you make a checkbox in HTML?

Use <input type="checkbox"> inside a form.

19. How do you add a submit button?

Use <input type="submit" value="Submit"> or <button type="submit">Submit</button> inside a form.

20. What is the difference between <div> and <span>?

<div> is a block-level element used to group larger sections, while <span> is an inline element used for small content parts.

21. What does the <iframe> tag do?

The <iframe> tag is used to embed another HTML page within the current page.

22. What is the use of the <fieldset> and <legend> tags?

<fieldset> groups related form elements, and <legend> provides a caption for the group.

23. What is the difference between id and class attributes?

id is unique and used to identify one element, while class can be used for multiple elements.

24. What is the purpose of the <script> tag?

The <script> tag is used to define client-side JavaScript code.

25. How do you add comments in HTML?

Use <!-- comment here --> to add comments.

26. What is the <noscript> tag?

The <noscript> tag provides fallback content if scripts are not supported or disabled.

27. How do you make a dropdown list in HTML?

Use the <select> tag with <option> tags.

28. What are data-* attributes?

data-* attributes are custom attributes used to store extra data private to the page or application.

29. How do you open a link in a new tab?

Use target="_blank" in the <a> tag.

30. What is the use of the <canvas> tag?

The <canvas> tag is used to draw graphics on the fly using JavaScript.

31. What is the difference between <section> and <div>?

<section> is a semantic tag representing a section of content, while <div> is a generic container.

32. How do you embed a video in HTML?

Use the <video> tag with src, controls, and optional attributes like autoplay or loop.

33. What is the difference between <em> and <i> tags?

<em> is used for emphasized text (semantic), <i> is for italicized text (presentational).

34. What are global attributes in HTML?

Global attributes can be used on any HTML element. Examples: id, class, style, title, lang.

35. What is the <base> tag?

The <base> tag specifies the base URL for all relative URLs in a document.

36. How do you specify an image map in HTML?

Use the <map> and <area> elements, and reference the map using the usemap attribute in the <img> tag.

37. What is the purpose of the <details> and <summary> tags?

They create a disclosure widget that users can open and close.

38. How can you include external CSS in HTML?

Use the <link> tag inside the <head> section: <link rel="stylesheet" href="style.css">.

39. What is the use of the <abbr> tag?

The <abbr> tag defines an abbreviation or acronym, optionally with a title attribute for full description.

40. How do you create a table in HTML?

Use the <table>, <tr>, <th>, and <td> elements to create a table structure.
